### 📂 本地包入口
如果您使用本地包，请参考以下路径格式：

* **主入口**: `file://包名/api.json`
* **Py 爬虫入口**: `file://包名/py.json`
* **Js 爬虫入口**: `file://包名/js.json`

> **注意**：以上 `file://` 开头的路径仅为配置示例，**在浏览器中无法直接点击打开**，请手动复制到 APP 配置中。
